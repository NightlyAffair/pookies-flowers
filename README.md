# Pookie's Flowers

An interactive website showcasing flower arrangements with scroll-based animations and 3D interactions.

## Getting Started

### Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the site.

## Tech Stack

See [TECH_STACK.md](./TECH_STACK.md) for detailed documentation on all technologies used in this project.

### Key Technologies
- Next.js 16 (React Framework)
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger (Animations)
- Three.js + React Three Fiber (3D Graphics)
- React Hook Form (Forms)

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components (create as needed)
- `public/` - Static assets

## Development Commands

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run linter
```

## Next Steps

1. Start the dev server with `npm run dev`
2. Edit `app/page.tsx` to begin building your homepage
3. Add components in a new `components/` directory
4. Add flower images to `public/` directory
5. Implement scroll animations with GSAP
6. Create 3D flower models with React Three Fiber
